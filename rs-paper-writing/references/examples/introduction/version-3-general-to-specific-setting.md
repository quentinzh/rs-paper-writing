# Version 3: General Task to Specific Setting

Use this when the general task is familiar but the paper studies a specific setting.

```text
Recommendation models typically learn user preferences from historical interactions.
This paper focuses on LLM-based sequential recommendation, where a user's history is converted into a natural-language prompt and the model generates the next recommended item.
```

Alternative:

```text
Debiasing recommendation aims to reduce systematic mismatch between user preference and recommendation exposure.
This paper focuses on popularity debiasing in generative recommendation, where an LLM may over-generate popular items during decoding.
```
