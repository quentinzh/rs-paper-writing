# Version 1: Task Then Application

Use this when the task is not obvious to all readers.

```text
Sequential recommendation aims to predict the next item a user may interact with based on the user's historical interaction sequence.
This task is important for personalized ranking, session continuation, and candidate generation in recommender systems.
```

Alternative:

```text
Item-side fairness in recommendation aims to avoid systematically over-exposing or under-exposing item groups in generated recommendation lists.
This objective is important for platforms where recommendation affects item visibility, creator exposure, and long-tail discovery.
```
