# Technical Challenge Version 1: Existing Task

Use when the task is established and the challenge is a known bottleneck.

```text
Next-item recommendation is challenging because user intent is shaped by both long-term preference and short-term context.
Recent sequential recommenders use attention-based encoders to model item transitions.
However, these methods may overfit frequent local patterns and under-represent sparse long-range interests, leading to weaker recommendations for users with diverse histories.
```

Reusable logic:

1. State the general challenge.
2. Name the mainstream mechanism.
3. State the remaining limitation.
4. Tie the limitation to recommendation quality.
