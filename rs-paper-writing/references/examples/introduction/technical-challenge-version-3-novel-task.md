# Technical Challenge Version 3: Novel Task

Use when the paper introduces a relatively new setting.

```text
In this work, we study confidence-aware generative recommendation, where the model must generate an item and also express how reliable the recommendation is.
This setting is challenging because recommendation feedback is sparse, user preference is stochastic, and generated confidence may not be directly calibrated to observed clicks.
Moreover, the model must produce confidence signals that are useful for ranking or reranking rather than merely reflecting token probability.
```

Reusable logic:

1. Define the new setting.
2. Explain why the setting is hard.
3. Clarify why existing assumptions are insufficient.
