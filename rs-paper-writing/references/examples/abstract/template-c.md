# Abstract Template C: Multiple Contributions

Use when the method has several necessary contributions.

```text
Recommendation under [setting] requires [requirement 1] and [requirement 2].
Existing methods improve [property], but they do not jointly address [problem 1] and [problem 2].
We propose [Name], a [framework] for [task].
First, [Name] introduces [contribution 1], which [advantage 1].
Second, [Name] designs [contribution 2], which [advantage 2].
Finally, [Name] uses [contribution 3] to [advantage 3].
Experiments on [datasets] demonstrate that [Name] achieves better [metric group] than [baseline group].
```

Reusable logic:

1. State the multi-requirement setting.
2. Explain why prior methods are incomplete.
3. Name the framework.
4. Pair each contribution with an advantage.
5. End with bounded evidence.
