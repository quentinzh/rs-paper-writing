# Abstract Template A: Challenge to Method

Use for one central recommendation challenge.

```text
Sequential recommendation aims to predict a user's next interaction from historical behavior and is central to personalized ranking.
Existing sequential recommenders usually encode user histories with attention-based architectures, but they often overemphasize frequent short-term patterns and fail to capture sparse long-term interests.
To address this problem, we propose [Model Name], a [model] for sequential recommendation.
[Model Name] combines [module 1] with [module 2] to represent both short-term intent and long-term preference.
Experiments on [datasets] show that [Model Name] outperforms strong sequential recommendation baselines on [metrics].
```

Reusable logic:

1. Define the task.
2. Name the existing paradigm.
3. State the exact challenge.
4. Propose the method.
5. State the evidence.
