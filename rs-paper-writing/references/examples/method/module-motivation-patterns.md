# Module Motivation Patterns

Use these openings to motivate RS method modules.

```text
A remaining challenge is that [existing method] treats all interactions as equally informative, although user feedback can be noisy or time-dependent.
```

```text
However, directly applying [LLM objective] to recommendation ignores [recommendation-specific property], leading to [failure].
```

```text
Existing debiasing methods usually operate at [training/inference] stage, but the observed bias also arises from [other stage].
```

```text
Although [baseline framework] improves [metric], it introduces [cost/validity/fairness issue], which limits its use in [setting].
```
