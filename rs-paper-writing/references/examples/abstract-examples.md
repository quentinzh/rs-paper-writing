# Abstract Examples Index

Use these templates only for sentence logic.

1. Challenge to method: `references/examples/abstract/template-a.md`
2. Diagnosis to method: `references/examples/abstract/template-b.md`
3. Multiple contributions: `references/examples/abstract/template-c.md`

Recommended choice:

1. Use Template A for a single recommendation modeling challenge.
2. Use Template B for LLM-based recommendation, fairness, debiasing, causal recommendation, or objective-level failure analysis.
3. Use Template C when the paper has multiple technical contributions that must be named separately.
